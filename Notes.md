**Python Notes**

***Functions***

print() function - prints what's in between parentheses(tells PC this is text)
input() function - "Enter string as prompt for user input"
str() function - converts int to string
len() function - finds the length of a string
type() function - tells what type of data a variable is
round() function - rounds - can specify decimal places after comma ie print(round(8/3), 2))
lower() - makes all letters in a string lowercase
count() - counts how many times letter occures in string
max() - highest value in a list
min() - lowest value in a list


***Data types***

string
integer
float
boolean

***Logical Operators***
and - both conditions must be true
or - one condition must be true or either is true
not - the reverse of whatever the operation is
